# Caixa de Conexão de Alta Amperagem

## Descrição

Caixa protetora para cobrir uma barra de cobre com parafusos, utilizada para conectar os fios de alimentação de alta potência do triciclo elétrico.

## Status

**Em desenvolvimento (WIP)** - Versão não finalizada

## Objetivo

- Proteger a barra de cobre e as conexões
- Isolar eletricamente os terminais de alta corrente
- Permitir fácil acesso para manutenção via parafusos
- Suportar alta potência sem superaquecimento
